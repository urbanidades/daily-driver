# Prompts

## Naming Convention
`[feature]-[agent].md` or `[task-id]-[description].md`

## Prompt Template

```markdown
# Context
[Link to relevant context files or summary of current state]

# Requirements
- [ ] Requirement 1
- [ ] Requirement 2

# Files to Reference
- [path/to/file1]
- [path/to/file2]

# Constraints
- Keep changes small.
- Follow the style guide.

# Handoff Notes
[Any specific instructions for the next agent]
```
